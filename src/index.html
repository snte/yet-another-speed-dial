3 ways to add speed dials:

1. Right-click on any page and select "Add to Speed Dial"

2. Bookmark a site and choose the Other Bookmarks > Speed Dial folder

3. Use the button below (it can be toggled on or off in the settings)

  • Open in new tab
  • Open in new window
  • Open in new private window
  • Edit
  • Delete
  • Settings...

This will temporarily open the site in a new tab to capture the image
